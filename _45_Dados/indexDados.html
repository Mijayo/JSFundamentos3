Lanzamiento de dos dados

Escribir un script que simule el lanzamiento de dos dados. Hacer uso de la función Math.random para obtener números
aleatorios entre 1 y 6 para cada uno de los lanzamientos de los dados. Sumar el resultado de lanzar dos dados
y anotar en un array el número de apariciones de dicha suma, repitiendo 36.000 veces esta operación.
